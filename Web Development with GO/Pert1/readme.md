# Student Portfolio Website
## Deskripsi
Project ini merupakan website portfolio sederhana yang dibuat menggunakan HTML, CSS, dan Go.
## Fitur
- Halaman About Me
- Daftar Skills
- Daftar Projects
- Halaman Contact
## Teknologi yang Digunakan
- Go (Golang)
- HTML
- CSS
## Struktur Project
```
student-portfolio/
├── main.go
├── readme.md
├── static
│   └── css
│       └── style.css
└── templates
    └── index.html
```
## Cara Menjalankan
1. Install Go terlebih dahulu
2. Jalankan perintah:
go run main.go
3. Buka browser dan akses:
http://localhost:8080
